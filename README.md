# CafeServerGo

A high-performance TCP socket server written in Go, ported from [Café Server Python](https://github.com/GGReborn/CafeServer), and Smartfox server.
Way more faster than any Python implementations.

## Features

- **TCP socket server** Handles concurrent client connections with Go's native concurrency model
- **Modular command system** Game logic is organized into independently registered command packages
